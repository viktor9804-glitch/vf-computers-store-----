# ВФ Компютри - V27 Real 10 Image Gallery Fix

Това е пълен fix за админ панела и магазина.

## Добавено
- Реален избор на до 10 снимки за един артикул.
- Първата снимка става основна.
- Всички снимки се записват в `images`.
- `image` остава за съвместимост и пази първата снимка.
- Продуктите показват миниатюри и gallery preview.

## Задължително в Supabase
В SQL Editor изпълни:

```sql
alter table products add column if not exists images jsonb default '[]'::jsonb;
```

## Много важно
Понеже при теб има отделно админ приложение:
- ако админът е в основния сайт: копирай `src/main.jsx` и `src/style.css`
- ако админът е във `vf-admin-app`: копирай и променените файлове от тази папка също

Най-сигурно:
1. Направи backup на проекта.
2. Копирай целия V27 проект върху стария, НО НЕ заменяй `api/`.
3. Изтрий `dist/`, `release/`, `node_modules/.vite` ако ги има.
4. Пусни `npm install`, `npm run build`.
5. За desktop app пусни build командата за Electron.

## Не заменяй
- `api/`, ако чат ботът работи.
