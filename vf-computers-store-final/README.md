# ВФ Компютри - V20 Partners Logo Style

## Какво е променено
- Партньорите вече имат отделни локални SVG лога
- Логата са в черно-червения VF Computers стил
- Няма външни image зависимости
- Картите остават кликаеми и отварят сайта на партньора

## Лога
Файловете са тук:

```txt
public/partners/
  polycomp.svg
  vali.svg
  adminbg.svg
  katnis13.svg
  tbibank.svg
  econt.svg
```

## Важно
Не е нужно да заменяш папка `api`, ако чат ботът вече работи.
За партньорите са нужни:
- `src/main.jsx`
- `src/style.css`
- `public/partners/`
